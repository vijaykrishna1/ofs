# Oracle Field Service – Simple Plugins

This repository contains simple sample plugins for Oracle Field Service built using the Plugin API framework.

The examples included here are intended for learning, experimentation, and quick reference when developing custom plugins.

## Technologies Used

The plugins in this repository are built using:

- Plain JavaScript
- HTML
- Oracle Field Service Plugin API

These samples are intentionally kept simple to help understand how plugins interact with Oracle Field Service.

They are shared for educational and demonstration purposes only.

Use them at your own discretion and ensure proper testing before using them in any production environment.
